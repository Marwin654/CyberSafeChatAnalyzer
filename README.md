# Cyber Safe Chat Analyzer

An AI-powered cybersecurity application that analyzes messages for:

- Spam Detection
- Toxicity Detection
- Risk Assessment

## Technologies

- Python
- Flask
- Scikit-Learn
- Pandas
- HTML/CSS

## Features

- Real-time message analysis
- Spam detection
- Toxicity detection
- Confidence scores
- Risk level evaluation

## Future Features

- Phishing detection
- Threat detection
- Dashboard analytics
